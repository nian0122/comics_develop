# 前端字体颜色重新设计方案

**日期**: 2026-05-10  
**作者**: Claude  
**状态**: 待实现  

## 背景

当前前端字体颜色存在严重的可读性问题，影响用户浏览体验。

### 发现的问题

1. **透明文字**: `--text-secondary: rgba(0, 0, 0, 0)` 完全不可见
2. **黑色文字**: `--text-tertiary/quaternary: rgb(0, 0, 0)` 在暗色背景下不可见
3. **对比度不足**: `--color-text-muted: #9aa3af` 对比度偏低
4. **变量混乱**: 新旧两套颜色系统混用，导致渲染不一致

## 设计目标

1. 修复当前颜色系统的 bug（透明/黑色文字）
2. 提升文字对比度，确保在各种设备上清晰可读
3. 保持暗色主题，适合长时间阅读漫画
4. 统一颜色变量，消除混乱

## 设计方案

### 采用方案：增强暗色主题

保持暗色基调，全面提升对比度和色彩层次。

### 颜色规范

#### 主色调

| 用途 | 旧值 | 新值 | 变化 |
|------|------|------|------|
| 主文字 | `#f1f3f5` | `#ffffff` | 提升为纯白 |
| 次要文字 | `rgba(0,0,0,0)` (透明) | `#c9d1d9` | 从透明修复 |
| 三级文字 | `rgb(0,0,0)` (黑色) | `#8b949e` | 从黑色修复 |
| 弱化文字 | `#9aa3af` | `#6e7781` | 调整层次 |

#### 强调色

| 用途 | 旧值 | 新值 | 变化 |
|------|------|------|------|
| 强调色 | `#8ea4ff` | `#58a6ff` | 更亮的科技蓝 |
| 强调悬停 | `#9eb4ff` | `#79b8ff` | 更亮的悬停态 |

#### 背景色（保持不变）

| 用途 | 值 |
|------|-----|
| 主背景 | `#0f1115` |
| 表面背景 | `#171a21` |
| 柔和背景 | `#20242d` |

### CSS 变量映射

```css
/* 新设计系统变量 - 增强暗色主题 */
:root {
    /* 文本色 - 修复对比度问题 */
    --color-text: #ffffff;
    --color-text-secondary: #c9d1d9;
    --color-text-tertiary: #8b949e;
    --color-text-muted: #6e7781;
    
    /* 强调色 - 更亮的蓝色 */
    --color-accent: #58a6ff;
    --color-accent-hover: #79b8ff;
    
    /* 向后兼容 - 旧变量名映射到新暗色主题 */
    --text-primary: rgba(255, 255, 255, 0.95);
    --text-secondary: #c9d1d9;      /* 修复：从透明 */
    --text-tertiary: #8b949e;       /* 修复：从黑色 */
    --text-quaternary: #6e7781;     /* 修复：从黑色 */
    
    --accent: #58a6ff;
    --accent-hover: #79b8ff;
    --accent-active: #388bfd;
    --accent-light: rgba(88, 166, 255, 0.15);
}
```

## 影响范围

### 需要修改的文件

1. **`frontend/css/variables.css`**
   - 更新所有文字颜色变量
   - 修复向后兼容的旧变量

2. **`frontend/css/components.css`**
   - 检查并更新组件样式中的颜色引用

3. **Vue 组件检查**
   - `SeriesPage.vue` - 系列名称、阅读进度提示
   - `DirectoryPage.vue` - 章节名称、目录层级
   - `ReaderShell.vue` - 控制按钮、进度显示
   - `ChapterCard.vue` - 章节卡片文字
   - `ToolsPage.vue` - 工具页面文字

## 可访问性

- 主文字对比度：白色 (#ffffff) 在 `#0f1115` 背景上 = **15.7:1** (WCAG AAA)
- 次要文字对比度：亮灰 (#c9d1d9) 在 `#171a21` 背景上 = **8.5:1** (WCAG AAA)
- 三级文字对比度：中灰 (#8b949e) 在 `#171a21` 背景上 = **5.2:1** (WCAG AA)
